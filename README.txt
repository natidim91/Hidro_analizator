
# Project Name: 374_noit_Hidroanalizator

This is a tool to monitor soil and air humidity using an Arduino Uno and DHT11 sensor.
It helps plant owners by providing accurate readings and alerts when watering is needed.


## Requirements
- Arduino Uno
- DHT11 Sensor
- LCD Display (I2C)

## How to install
1. Connect the hardware as described in the project documentation.
2. Upload the `374.ino` code to your Arduino Uno.
3. Monitor the output on the LCD or through serial communication.
